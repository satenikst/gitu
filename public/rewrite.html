git commit --amend

git status հրամանը ցուցադրում է աշխատանքային միջավայրի վիճակը։ Սա քեզ թույլ է տալիս տեսնել, թե ինչպիսի փոփոխություններ են արվել, ինչպիսիք չեն արվել։ Ինչպես նաև ցույց է տալիս, թե Git-ի կողմից ինչպիսի ֆայլեր են անտեսվել։

git status-ը չի ցուցադրում որևէ ինֆորմացիա կապված քո նախագծի պատմության հետ։ Դրա համար դու պետք է օգտագործես git log հրամանը։

git log հրամանը ցուցադրում է տեղի ունեցած բոլոր commit-ները։ Դա քեզ թույլ է տալիս թերթել նախագծի պատմության էջերը, զտել (filter) այն, և փնտրել հատուկ փոփոխություններ։

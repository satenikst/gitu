git remote

git remote հրամանի միջոցով կարող ես ստեղծել, դիտել, անվանափոխել և ջնջել այլ շտեմարանների հետ կապերը։ Հեռակա կապերը (remote connections) ավելի շատ նման են էջանշանների (bookmarks), քան ուղղակի հղումների (դեպի այլ շտեմարաններ)։ Դեպի այլ շտեմարաններ մուտք ապահովելու փոխարեն, դրանք ծառայում են որպես հարմար անուններ։

կիրառում
git remote

Սա ցույց է տալիս քո ունեցած բոլոր հեռակա կապերը։

git remote -v

Ցույց է տալիս կապերի ամբողջական URL հղումները։

git remote add 'name' 'url'

Ստեղծում է նոր կապ դեպի հեռակա շտեմարան։

git remote rm 'name'

Ջնջում է 'name' կոչվող հեռակա կապը։

git remote rename 'old-name' 'new-name'

Անվանափոխում է հեռակա կապը 'old-name'-ից 'new-name'։

git fetch

git fetch հրամանը ներմուծում է commit-ները հեռակա շտեմարանից դեպի լոկալ աշխատանքային միջավայր։ Ստացված commit-ները պահպանվում են որպես հեռակա ճյուղեր, նորմալ, լոկալ ճյուղերի փոխարեն, որոնց հետ մենք աշխատել ենք։ Սա քեզ հնարավորություն է տալիս դիտելու կատարված փոփոխությունները, քո նախագծի կրկնօրինակի մեջ դրանք ինտեգրելուց առաջ։

կիրառում
git fetch 'remote'

Ստանում է շտեմարանի բոլոր ճյուղերը։ Սա նույնպես այլ շտեմարանից վերբեռնում է բոլոր պահանջվող commit-ներն ու ֆայլերը։

Պահեստի կարգավորումներ

git init

git init հրամանը ստեղծում է նոր Git շտեմարան։ Սա օգտագործվում է արդեն գոյություն ունեցող Git շտեմարանը փոխելու կամ նոր Git շտեմարան ստեղծելու համար։ Այսպիսով, սա առաջին հրամանն է, որը դու կօգտագործես նոր նախագիծ ստեղծելիս։ git init-ը նախագծի հիմքում ստեղծում է .git ենթացուցակ, որն իր մեջ պարունակում է բոլոր կարևոր տվյալները։

git init

git clone

git clone հրամանը կրկնօրինակում է (copy) արդեն գոյություն ունեցող git շտեմարանը։ clone-ը ավտոմատ կերպով ստեղծում է հեռակա կապ դեպի օրիգինալ շտեմարան։ Սրա միջոցով հնարավոր է դառնում հեշտությամբ համագործակցել կենտրոնական շտեմարանի հետ։

կիրառում
git clone

Շտեմարանի կրկնօրինակը գտնվում է քո լոկալ PC-ի մեջ։ Օրիգինալ շտեմարանը կարող է տեղակայված լինել լոկալ ֆայլային համակարգի վրա կամ հեռակա, որը հասանելի է HTTP-ի կամ SSH-ի միջոցով։

git init

git init հրամանը ստեղծում է նոր Git շտեմարան։ Սա օգտագործվում է արդեն գոյություն ունեցող Git շտեմարանը փոխելու կամ նոր Git շտեմարան ստեղծելու համար։ Այսպիսով, սա առաջին հրամանն է, որը դու կօգտագործես նոր նախագիծ ստեղծելիս։ git init-ը նախագծի հիմքում ստեղծում է .git ենթացուցակ, որն իր մեջ պարունակում է բոլոր կարևոր տվյալները։

git clone

The git clone command copies an existing Git repository. This is sort of like svn checkout, except the “working copy” is a full-fledged Git repository—it has its own history, manages its own files, and is a completely isolated environment from the original repository. As a convenience, cloning automatically creates a remote connection called origin pointing back to the original repository. This makes it very easy to interact with a central repository.
